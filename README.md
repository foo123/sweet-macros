sweet-macros
============

various sweet.js macros for javascript

1. **Class** macro, see [classy.js](https://github.com/foo123/classy.js) 


2. **ES6 function** with default arguments and rest arguments

**Example:**

```javascript

function afunc(arg1=123, arg2=null, ...rest)
{
    // func body etc..
}

```
